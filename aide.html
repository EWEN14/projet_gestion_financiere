Inscription

Lors de la première connexion vous devez vous inscrire pour accéder à l’application.

Pour cela cliquez sur le lien dans le menu du haut « inscription » (cf. capture ci-contre).

Vous serez redirigé vers la page d’inscription à l’application (cf. capture ci-dessous).

Saisissez un identifiant et un mot de passe de votre choix. Il vous est demandé de saisir deux fois votre mot de passe afin d’éviter les erreurs de saisie, puis cliquez sur le bouton « Valider ».

Vous serez alors redirigé sur la page d’accueil pour vous identifier (cf. capture ci-dessous).

Authentification

Pour accéder à l’application vous devez vous identifier.

Pour cela, saisissez l’identifiant et le mot de passe que vous avez saisis lors de votre inscription (cf. capture ci-dessous).

Puis cliquez sur le bouton « Connexion ».

Vous serez redirigé sur la page d’accueil.

Les calculs


Calcul de seuil de rentabilité

Pour effectuer un calcul, saisissez le « Prix de vente… », le « Chiffe d’Affaires », les « Charges Variables » et les « Charges Fixes »

Puis cliquez sur le bouton « Calculer ».

Ce dernier devient inactif lorsque le calcul a été effectué.

SAUVEGARDE

Pour sauvegarder votre calcul, cliquez sur le bouton :

Ce dernier devient inactif lorsque la sauvegarde a été effectuée.

Pour accéder aux calculs sauvegardés, cliquez sur l’onglet « Calculs Sauvegardés »

Calcul de coût unitaire pondéré


STOCK INITIAL

Vous devez en premier saisir un stock initial, puis cliquez sur le bouton de validation pour enregistrer la ligne.

S'il n'y a pas de stock initial, saisissez des 0 pour chaque valeur.

ENTRÉE

Pour saisir une entrée vous devez cliquer sur le bouton « + Nouvelle entrée ».

Une ligne de saisie apparaît.

Saisissez la « Quantité » et le « Coût Unitaire ».

Puis validez la ligne.

Valider la ligne saisie.

Supprimer la ligne en cours de saisie ou la dernière ligne saisie

Si vous cliquez sur l’un des boutons pour saisir une nouvelle ligne, le message suivant apparaîtra pour vous signalez que vous devez sortir de la saisie soit en validant soit en supprimant la ligne actuelle.

SORTIE

Pour saisir une sortie vous devez cliquer sur le bouton « + Nouvelle sortie ».

Une ligne de saisie apparaît.

Saisissez la « Quantité » et le « Coût Unitaire ».

RÉINITIALISATION DU TABLEAU

Pour réinitialiser le tableau cliquez sur le bouton « Réinitialiser »

Changement de mot de passe

Pour changer votre mot de passe cliquez sur « Mot de passe oublié » dans le menu du haut.

Il vous est demandé de saisir deux fois votre nouveau mot de passe pour éviter les erreurs de saisie.

Puis cliquez sur le bouton « Valider ».
