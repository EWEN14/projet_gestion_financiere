Inscription

Pour accéder à l’application, lors de la première connexion, l’inscription est nécessaire.

Pour cela, cliquez sur le lien Inscription dans le menu situé en haut à gauche.

Après avoir été redirigé vers la page d’inscription de l’application, entrez un identifiant et un mot de passe.

Afin d’éviter une quelconque erreur de saisie, une confirmation du mot de passe sera demandée.

Si tous les champs de saisies ont été remplis, cliquez sur le bouton Valider.

Après avoir appuyé sur Valider, vous êtes redirigé sur la page d’accueil.

Authentification

Afin d’accéder à l’application, saisir l’identifiant et le mot de passe, créés lors de l’inscription, dans les champs de saisies.

Puis cliquez sur le bouton Connexion. Vous serez, alors, redirigé sur la page d’accueil.

Accueil

La page d’accueil affiche Vos derniers calculs, soit les calculs sauvegardés lors de la dernière connexion.

Calculs


Calcul de seuil de rentabilité

Pour effectuer un calcul, saisir dans la colonne Calcul le Prix de Vente HT du produit (PV), le Chiffre d’Affaires (CA), les Charges Variables (CV) et les Charges Fixes (CF). Cliquer sur le bouton Calculer.

Lorsque le calcul a bien été effectué, le bouton Calculer devient inactif.

Sauvegarde

Pour sauvegarder un calcul, cliquer sur le bouton Sauvegarder ce calcul.

Lorsque la sauvegarde a bien été effectuée, le bouton Sauvegarder ce calcul devient inactif.

Pour accéder aux calculs sauvegardés, cliquer sur l’onglet Calculs Sauvegardés.

Calcul de coût unitaire moyen pondéré


Stock initial

Dans un premier temps, il faut saisir la Quantité et le Coût Unitaire du Stock Initial. Pour un stock initial nul, saisir la valeur de 0 dans chaque champ. Pour enregistrer la ligne de stock initial, cliquer sur le bouton vert avec l’icône valider.


Nouvelle Entrée

Pour saisir une entrée, cliquer sur le bouton + Nouvelle entrée. Une ligne Entrée apparait. Saisir dans la colonne Entrée, la Quantité et le Coût Unitaire de la ligne.

Il est possible de valider la ligne Entrée en cliquant sur le bouton vert avec l’icône valider ou supprimer la ligne Entrée en cliquant sur le bouton rouge avec l’icône poubelle.

Si les champs, Quantité et Coût Unitaire, de la colonne Entrée, non pas été complétés par des valeurs, la fenêtre nommée localhost indique signale le message suivant :

« Veuillez remplir le(s) champ(s) avant de valider. »


Sortie

Pour saisir une sortie, cliquer sur le bouton + Nouvelle sortie. Une ligne Sortie apparait. Saisir dans la colonne Sortie, la Quantité et le Coût Unitaire de la ligne.

Il est possible de valider la ligne Sortie en cliquant sur le bouton vert avec l’icône valider ou supprimer la ligne Entrée en cliquant sur le bouton rouge avec l’icône poubelle.

Si les champs, Quantité et Coût Unitaire, de la colonne Sortie, non pas été complétés par des valeurs, la fenêtre nommée localhost indique signale le message suivant :

« Veuillez remplir le(s) champ(s) avant de valider. »


Réinitialisation du tableau

Pour réinitialiser le tableau cliquez sur le bouton Réinitialiser.

Changement de mot de passe

Pour changer votre mot de passe cliquez sur Password dans le menu du haut.

Afin d’éviter une quelconque erreur de saisie, une confirmation du mot de passe sera demandée. Cliquer sur le bouton Valider.
